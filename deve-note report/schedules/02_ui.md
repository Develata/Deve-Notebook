# 前端交互进度 (Frontend Schedule)

> 涵盖计划: 03_rendering, 08_ui_design, 10_i18n, 12_commands, 13_settings

## 1. 渲染引擎 (Rendering - Plan 03)
- [x] **Markdown Engine**: CodeMirror 6 基础集成.
- [x] **Interaction Philosophy**:
    - [x] **Source-First**: 源码优先，装饰为辅.
    - [x] **Cursor Reveal**: 光标接触时隐藏渲染组件.
- [x] **Core Capabilities**:
    - [x] **Math**: KaTeX 集成，支持 Inline `$`/Block `$$`.
    - [x] **Mermaid**: 代码块自动渲染，高度自适应.
    - [x] **Smart Tables**: Markdown 表格转 HTML Table，支持样式.
    - [x] **Interactive Tasks**: `[ ]` Checkbox 双向绑定.
    - [x] **Block Styling**: 代码块/引用块全行背景色.
- [x] **Nested Hell**: 支持多层级引用/列表嵌套样式 (`depth` 变量).
- [x] **Code Block Toolbar**: 
    - [x] **Copy Button**: 一键复制并反馈.
    - [x] **Menu Button**: 扩展菜单入口 (预留接口).
    - [x] **Position**: 悬浮于代码块右上角 (Absolute).
