# 15_release.md - 开源发布篇 (Release Strategy)

## 开源与发布 (Open Source & Release)

*   **许可证**：双协议许可证 MIT / Apache-2.0。
*   **发行物**：
    *   **Desktop**: Tauri bundle (msi/dmg/deb)。
    *   **Docker**: GHCR 镜像, 支持数据卷挂载。
    *   **Web**: PWA (Progressive Web App).
*   **GitHub Releases**：提供二进制与校验和 (Checksums)。
*   **插件发布**：官方索引，支持离线安装。
